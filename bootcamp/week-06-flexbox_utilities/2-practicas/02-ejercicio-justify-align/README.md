# Ejercicio 02: Justify y Align

## 🎯 Objetivo

Practicar todas las variantes de `justify-*` e `items-*` para distribuir y alinear ítems en ambos ejes de Flexbox, y usar `self-*` para alinear ítems individuales.

---

## 📋 Instrucciones

### Paso 1: Variantes de `justify-*`

Descomenta el **bloque 1** para ver una galería visual de las 6 variantes de `justify-content`: `justify-start`, `justify-center`, `justify-end`, `justify-between`, `justify-around` y `justify-evenly`.

### Paso 2: Variantes de `items-*`

Descomenta el **bloque 2** para ver las 5 variantes de `align-items` en acción: `items-start`, `items-center`, `items-end`, `items-stretch` e `items-baseline`. Cada contenedor tiene altura fija para que el efecto sea visible.

### Paso 3: Centrado perfecto

Descomenta el **bloque 3** para ver el patrón `flex items-center justify-center` aplicado a una tarjeta de login centrada en la pantalla.

### Paso 4: `self-*` para ítems individuales

Descomenta el **bloque 4** para ver cómo `self-start`, `self-center`, `self-end` y `self-stretch` permiten que ítems individuales ignoren el `items-*` del contenedor.

---

## ✅ Verificación

- El bloque 1 muestra claramente la diferencia visual entre las 6 distribuciones `justify-*`
- El bloque 2 muestra los ítems alineados en diferentes posiciones del eje cruzado
- El bloque 3 muestra la tarjeta perfectamente centrada horizontal y verticalmente
- El bloque 4 muestra ítems con alineaciones diferentes dentro del mismo contenedor
