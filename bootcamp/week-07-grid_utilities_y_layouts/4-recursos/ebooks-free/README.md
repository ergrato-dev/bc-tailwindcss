# 📚 Ebooks y Recursos Gratuitos — Semana 7

## 📖 Guías Interactivas (altamente recomendadas)

### 1. An Interactive Guide to CSS Grid — Josh Comeau
- **Tipo**: Artículo interactivo extenso (equivale a un ebook)
- **URL**: https://www.joshwcomeau.com/css/interactive-guide-to-grid/
- **Tiempo estimado**: 45-60 minutos
- **Por qué leerlo**: La mejor introducción visual a Grid. Cada concepto incluye demos interactivos donde puedes modificar el código en tiempo real. Cubre `auto-fill`, `minmax()`, `place-items` y mucho más.

### 2. CSS Grid Garden (si no lo has completado)
- **Tipo**: Juego educativo (28 niveles)
- **URL**: https://cssgridgarden.com/
- **Tiempo estimado**: 30-45 minutos (niveles 1-28)
- **Por qué jugarlo**: La forma más eficaz de memorizar las propiedades de posicionamiento Grid (`grid-column`, `grid-row`, `grid-area`).

## 📄 Documentación de Referencia Rápida

### 3. CSS-Tricks: Complete Guide to Grid (en formato cheatsheet)
- **URL**: https://css-tricks.com/snippets/css/complete-guide-grid/
- **Formato**: Tabla de referencia de todas las propiedades Grid
- **Uso recomendado**: Consulta rápida durante los ejercicios y el proyecto

### 4. Grid by Example — Rachel Andrew
- **URL**: https://gridbyexample.com/
- **Descripción**: Colección de patrones de layout Grid con ejemplos de código. Creado por Rachel Andrew, una de las personas que llevó CSS Grid al W3C.
- **Sección recomendada**: Gallery → ejemplos de layouts magazine, sidebar y dashboard

## 🗒️ Notas de Estudio Recomendadas

Al terminar esta semana, crea un documento personal con:

```
## CSS Grid — Mis notas

### Cuándo uso Grid:
- [ ] layouts de página (header/sidebar/main/footer)
- [ ] galerías de cards/imágenes
- [ ] pricing, features, testimonios

### Cuándo uso Flex:
- [ ] navbar
- [ ] botones con iconos
- [ ] listas de items inline

### Clases que más uso:
- grid grid-cols-3 gap-6
- grid-cols-1 md:grid-cols-2 lg:grid-cols-3
- col-span-2 / col-span-full
- grid-cols-[1fr_2fr] / grid-rows-[auto_1fr_auto]
- grid-cols-[repeat(auto-fill,minmax(250px,1fr))]
```
