## HTML y CSS Esencial

### 📘 Dive Into HTML5
- **Autor**: Mark Pilgrim
- **Enlace**: [https://diveintohtml5.info/](https://diveintohtml5.info/)
- **Qué encontrarás**: Historia y filosofía de HTML5, semántica, accesibilidad
- **Nivel**: Principiante - Intermedio
- **Secciones recomendadas para esta semana**: "What Does It All Mean?" (semántica), "Detecting HTML5 Features"

### 📘 CSS Notes for Professionals
- **Fuente**: GoalKicker.com (documentación Stack Overflow compilada)
- **Enlace**: [https://books.goalkicker.com/CSSBook/](https://books.goalkicker.com/CSSBook/)
- **Qué encontrarás**: Referencia completa de CSS con ejemplos prácticos
- **Nivel**: Principiante - Avanzado
- **Secciones recomendadas**: Box Model, Selectors, Specificity, Units

### 📘 HTML Notes for Professionals
- **Fuente**: GoalKicker.com
- **Enlace**: [https://books.goalkicker.com/HTMLBook/](https://books.goalkicker.com/HTMLBook/)
- **Qué encontrarás**: Referencia práctica de HTML con snippets listos para usar
- **Nivel**: Principiante - Intermedio
