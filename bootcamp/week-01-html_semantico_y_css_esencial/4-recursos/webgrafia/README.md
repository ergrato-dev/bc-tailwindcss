## Herramientas Online

### 🛠️ W3C HTML Validator
- **URL**: [https://validator.w3.org/](https://validator.w3.org/)
- **Para qué**: Valida que tu HTML sea correcto según el estándar
- **Uso**: Pega tu código o una URL y obtén errores/advertencias

### 🛠️ CSS Specificity Calculator
- **URL**: [https://specificity.keegan.st/](https://specificity.keegan.st/)
- **Para qué**: Calcula visualmente la especificidad de cualquier selector CSS

### 🛠️ Coolors Contrast Checker
- **URL**: [https://coolors.co/contrast-checker](https://coolors.co/contrast-checker)
- **Para qué**: Verifica si un par de colores cumple con el contraste WCAG AA (4.5:1)

### 🛠️ WAVE Web Accessibility Evaluator
- **URL**: [https://wave.webaim.org/](https://wave.webaim.org/)
- **Para qué**: Auditoría de accesibilidad de páginas web en tiempo real

### 🛠️ Can I Use
- **URL**: [https://caniuse.com/](https://caniuse.com/)
- **Para qué**: Verifica la compatibilidad de propiedades CSS/HTML en distintos browsers

### 🛠️ CSS Grid Garden / Flexbox Froggy
- **CSS Grid**: [https://cssgridgarden.com/](https://cssgridgarden.com/)
- **Flexbox**: [https://flexboxfroggy.com/#es](https://flexboxfroggy.com/#es)
- **Para qué**: Juegos interactivos para aprender Grid y Flexbox (serán útiles en Semana 6)
