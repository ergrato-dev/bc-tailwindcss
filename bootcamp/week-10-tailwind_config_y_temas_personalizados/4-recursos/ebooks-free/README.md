# Ebooks y Lecturas — Semana 10

> Material de lectura recomendado para profundizar en design systems, tipografía y CSS avanzado.

---

## 📗 Gratuitos / Free Tier

### Refactoring UI — Free Chapters
**Autores**: Adam Wathan y Steve Schoger (los creadores de Tailwind)
**Dónde**: [refactoringui.com](https://www.refactoringui.com)
**Qué leer**: El capítulo de *Color* explica exactamente cómo construir paletas con escala de luminosidad — el mismo sistema que usa Tailwind.

---

### Every Layout — Free Layouts
**Autores**: Andy Bell y Heydon Pickering
**Dónde**: [every-layout.dev](https://every-layout.dev)
**Qué leer**: Sección *Stack* y *Sidebar* — layouts primitivos que aplican spacing semántico como el que configuramos con `theme.extend.spacing`.

---

### CSS for JavaScript Developers — Free Module
**Autor**: Josh Comeau
**Dónde**: [css-for-js.dev](https://css-for-js.dev)
**Qué leer**: El módulo gratuito de *Fundamentals* cubre el cascade, variables y especificidad desde el modelo mental correcto.

---

### MDN — CSS Custom Properties (Variables)
**Autores**: Colaboradores de MDN / Mozilla
**Dónde**: [developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)
**Qué leer**: Todo el artículo — es la referencia más completa sobre CSS variables con ejemplos de herencia, fallbacks y manipulación con JS.

---

### Tailwind CSS Documentation
**Autores**: Tailwind Labs
**Dónde**: [tailwindcss.com/docs](https://tailwindcss.com/docs)
**Qué leer**:
- [Theme Configuration](https://tailwindcss.com/docs/theme)
- [Adding Custom Styles](https://tailwindcss.com/docs/adding-custom-styles)
- [Customizing Colors](https://tailwindcss.com/docs/customizing-colors)

---

## 📘 De Pago (Muy Recomendados)

### Refactoring UI — Libro Completo
**Autores**: Adam Wathan y Steve Schoger
**Precio**: ~$99 USD
**Por qué vale la pena**: El único libro de diseño UI escrito por un ingeniero. Cada concepto tiene código. El capítulo de color y tipografía cambiarán cómo ves el diseño para siempre.

---

### Design Systems Handbook
**Autores**: Marco Suarez et al. — InVision Design
**Dónde**: Disponible en [designbetter.co](https://www.designbetter.co/design-systems-handbook) (registro gratuito)
**Qué leer**: Capítulos 3 y 4 — *Design tokens* y *Building a design system* — contexto profesional de lo que hacemos en este bootcamp.

---
