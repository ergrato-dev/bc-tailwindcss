# Recursos adicionales — Semana 08

## 📚 Ebooks y guías gratuitas

### Lectura recomendada

| Recurso | Tipo | Descripción |
|---------|------|-------------|
| [CSS for JavaScript Developers — Animations Module](https://css-for-js.dev/) | Curso/guía interactiva | Josh Comeau. Módulo gratuito sobre animaciones y transiciones CSS. Fundamentos sólidos detrás de `transition-*` de Tailwind. |
| [Every Layout](https://every-layout.dev/) | Guía gratuita | Patrones de layout CSS primitivos. Útil para entender las decisiones de diseño detrás de flex y grid. |
| [Inclusive Components](https://inclusive-components.design/) | Guía gratuita | Patrones de componentes UI accesibles. Covers navbars, cards, buttons y más con foco en accesibilidad real. |
| [Accessible Rich Internet Applications (ARIA) Guide](https://www.w3.org/WAI/ARIA/apg/) | Documentación W3C | Patrones WAI-ARIA para componentes interactivos. Referencia cuando necesites implementar aria-expanded, aria-controls, etc. |
| [The Component Gallery](https://component.gallery/) | Referencia visual | Catálogo de cómo distintos design systems implementan componentes comunes: botones, cards, badges, navbars. |
