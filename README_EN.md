## 📋 Description

An intensive **12-week (~3 months)** bootcamp focused on mastering **TailwindCSS** and modern UI design. Designed to take students from zero to **Junior Frontend Developer**, emphasizing the utility-first philosophy, design systems, and real-world projects.

### 🎯 Goals

By the end of the bootcamp, students will be able to:

- ✅ Master the HTML semantics and CSS fundamentals needed to use Tailwind
- ✅ Apply the utility-first philosophy to build modern interfaces
- ✅ Use Tailwind's design system (colors, typography, spacing)
- ✅ Build complex layouts with Flexbox and Grid utilities
- ✅ Implement responsive design with a mobile-first approach
- ✅ Design reusable UI components (navbar, cards, forms, modals)
- ✅ Customize Tailwind with `tailwind.config.js` and custom themes
- ✅ Use official plugins (Typography, Forms, Aspect Ratio, Line Clamp)
- ✅ Implement dark mode, animations, and transitions
- ✅ Integrate TailwindCSS with React, Next.js, and component libraries

### 🚀 Why TailwindCSS?

> **Modern TailwindCSS from day one** — No legacy classes, only current best practices with Tailwind v4+.

TailwindCSS is the world's most popular utility-first CSS framework. This bootcamp focuses exclusively on Tailwind v4+ and modern frontend tooling. Students learn the exact techniques they will use in a professional environment.

## 🗓️ Bootcamp Structure

|           Stage                | Weeks | Hours | Main Topics                                                    |
| :----------------------------: | :---: | :---: | -------------------------------------------------------------- |
| **Web Fundamentals**           |  1-2  |  16h  | Semantic HTML, essential CSS, dev environment, intro Tailwind v4 |
| **Core Tailwind**              |  3-5  |  24h  | Colors, typography, spacing, borders, shadows, interactive states |
| **Layout**                     |  6-7  |  16h  | Flexbox utilities, Grid utilities, real-world layouts          |
| **Components & UI**            |  8-9  |  16h  | Navbar, cards, buttons, forms, modals, alerts                  |
| **Advanced Tailwind**          | 10-11 |  16h  | Config, custom themes, plugins, animations, dark mode          |
| **Ecosystem & Production**     |  12   |   8h  | React/Next.js, daisyUI, shadcn/ui, optimization, final project |

**Total: 12 weeks** | **96 hours** of intensive training

## 🚀 Quick Start

### Prerequisites

- **Node.js 22+** installed ([nodejs.org](https://nodejs.org/))
- **pnpm** installed (`npm install -g pnpm`)
- **Git** for version control
- **VS Code** (recommended) with included extensions
- Modern browser (Chrome, Firefox, Edge)

### 1. Clone the Repository

```bash
git clone https://github.com/ergrato-dev/bc-tailwindcss.git
cd bc-tailwindcss
```

### 2. Install VS Code Extensions

```bash
# Open in VS Code
code .

# Recommended extensions will appear automatically
# Or run: Ctrl+Shift+P → "Extensions: Show Recommended Extensions"
```

### 3. Navigate to the Current Week

```bash
cd bootcamp/week-01-html_semantico_y_css_esencial
```

### 4. Follow the Instructions

Each week contains a `README.md` with detailed instructions.

## 📊 Learning Methodology

### Teaching Strategies

- 🎯 **Project-Based Learning (PBL)**
- 🧩 **Deliberate Practice**
- 🎨 **UI Challenges** (replicas of real interfaces)
- 👥 **Peer Code Review**
- 🎮 **Live Coding**

### Time Distribution (8h/week)

- **Theory**: 2–2.5 hours
- **Practices**: 3–3.5 hours
- **Project**: 2–2.5 hours

### Evaluation

Each week includes three types of evidence:

1. **Knowledge 🧠** (30%): Quizzes and theoretical assessments
2. **Performance 💪** (40%): In-class practical exercises
3. **Product 📦** (30%): Evaluated deliverables (functional projects)

**Passing criteria**: Minimum 70% in each type of evidence
